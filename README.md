# Football Management System (Web Project)

This is a **web-based Football Management System** developed using **HTML, CSS, JavaScript, and SQL**.  
It allows users to manage football teams, players, and match data through a user-friendly web interface.  
The system demonstrates **frontend and backend integration**, **database management**, and **CRUD operations** (Create, Read, Update, Delete).
